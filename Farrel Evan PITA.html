fasta36 -H -E 1 -n -Q -m 9 all_PGMY_050312_seqs.fasta all_references.fasta

Now that the data has been read in. We will cluster. There are two primers used in analysis, pg5 and pgmy11. Each has their own file which needs to be loaded and then clustered separately. The results of each primer was used to confirm the presenceof each organism via two different methodologies.check with evan
